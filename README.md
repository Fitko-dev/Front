# Front
This app has been created using React Native Release.
Please follow these steps to set your React Native Environment:
  1)Download Node.Js
  2)Download Android Studio and Install an Emulator within
  3)Download Expo-cli
  4)Download Yarn
  5)Download Java JDK
  6)Download React Native
  7)Download Git
